## Hospedagem

Este site está pronto para hospedagem em qualquer servidor estático:

### Opções de Hospedagem Gratuita:
- **Netlify**: Arraste a pasta para netlify.com/drop
- **Vercel**: vercel.com
- **GitHub Pages**: Faça upload para um repositório
- **Cloudflare Pages**: pages.cloudflare.com

### Servidor Local (para teste):
```bash
# Com Python
python -m http.server 8000

# Com Node.js
npx serve .
```

## Recursos Externos

O site utiliza CDNs para:
- **Fontes**: Google Fonts (Oswald, Inter)
- **Ícones**: Lucide Icons (CDN)
- **Imagens**: Hospedadas em emergentagent.com
